## <a name="adding-a-database-provider-to-your-application"></a><span data-ttu-id="7234f-224">向应用程序添加数据库提供程序</span><span class="sxs-lookup"><span data-stu-id="7234f-224">Adding a database provider to your application</span></span>

<span data-ttu-id="7234f-225">EF Core 的大多数数据库提供程序都是作为 NuGet 包分发的。</span><span class="sxs-lookup"><span data-stu-id="7234f-225">Most database providers for EF Core are distributed as NuGet packages.</span></span> <span data-ttu-id="7234f-226">这意味着可使用命令行中的 `dotnet` 工具来安装它们：</span><span class="sxs-lookup"><span data-stu-id="7234f-226">This means they can be installed using the `dotnet` tool in the command line:</span></span>

<span data-ttu-id="7234f-227">或者在 Visual Studio 中，使用 NuGet 包管理器控制台：</span><span class="sxs-lookup"><span data-stu-id="7234f-227">Or in Visual Studio, using NuGet's Package Manager Console:</span></span>

``` powershell
install-package provider_package_name
```

<span data-ttu-id="7234f-228">安装后，需采用 `OnConfiguring` 方法或 `AddDbContext` 方法（如果使用的是依赖关系注入容器）在 `DbContext` 中配置提供程序。</span><span class="sxs-lookup"><span data-stu-id="7234f-228">Once installed, you will configure the provider in your `DbContext`, either in the `OnConfiguring` method or in the `AddDbContext` method if you are using a dependency injection container.</span></span>

<span data-ttu-id="7234f-230">数据库提供程序可扩展 EF Core，启用特定数据库特有的功能。</span><span class="sxs-lookup"><span data-stu-id="7234f-230">Database providers can extend EF Core to enable functionality unique to specific databases.</span></span>
