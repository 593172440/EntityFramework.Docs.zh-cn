## <a name="imodelcachekeyfactory"></a>IModelCacheKeyFactory
但是，如果尝试执行上述操作而不进行其他更改，则为 `IgnoreIntProperty` 的任何值创建新上下文时都会获得相同的模型。 这是由 EF 使用的模型缓存机制导致的，EF 使用此机制通过仅调用模型 `OnModelCreating` 一次并且缓存模型来提高性能。
